## Metric Ranks


In this view, 2D combinations are ranked from 1 to N for each metric, with 1 corresponding to the best combination (i.e. the one with the highest orthogonality value for that metric).
Decimal ranks (e.g. 5.5) indicate that midranks were applied to tied combinations. 
In this case, combinations with identical values are assigned the average of the ranks they would otherwise occupy.
