## Elution Domain

Represents the percentage of a common theoretical maximum 2D elution-composition space covered by a combination.

It is calculated as:

Elution Domain (%) = **(ΔΦe,1 × ΔΦe,2) / 100**

Where **ΔΦe,1 and ΔΦe,2** are the elution-composition ranges in the first and second dimensions.

A value of **100%** corresponds to the theoretical reference space of **100 × 100**. Higher values therefore indicate a larger covered elution-composition domain.

This comparison is made within a **global practical framework**, assuming that a mode able to exploit only a limited portion of the global reference space also provides a more limited composition domain in practice.
