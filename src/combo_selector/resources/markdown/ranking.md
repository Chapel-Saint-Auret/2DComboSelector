# Ranking

Controls how 2D combinations are ordered in the result table.

## Ranking based on

| Option | Description |
|--------|-------------|
| **Suggested score** | Uses the pre-computed orthogonality score recommended by the tool. Best for most users. |
| **Computed score** | Uses the score you calculated in the *Orthogonality assessment* step. |

## Tips

- If no computed score is available, only **Suggested score** will produce results.
- You can switch between modes at any time and re-run the ranking.
